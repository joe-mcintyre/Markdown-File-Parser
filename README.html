Markdown File Compiler

Markdown File Compiler is a C-based tool that converts Markdown files into HTML and hosts them using a local web socket.

Features

  • Markdown to HTML conversion
  • Automatic styling
  • Local web hosting
  • Lightweight and fast

Installation

  1. Clone the repository:
sh git clone https://github.com/your-username/markdown-file-Compiler.git cd markdown-file-Compiler
  1. Compile the program:
sh make

Usage

  1. Run the compiler:
sh // not functional yet
  1. Open a browser and visit:
http://localhost:8080

Requirements

- GCC Compiler - Make - Basic C Libraries
